## 🔧 步骤 1：安装 Git（5分钟）

### 1.1 下载 Git

**官方下载**：https://git-scm.com/download/win

**直接下载链接**（64位）：
```
https://github.com/git-for-windows/git/releases/download/v2.43.0.windows.1/Git-2.43.0-64-bit.exe
```

### 1.2 安装 Git

1. 双击下载的安装程序
2. **全部选择默认选项**
3. 一直点击 **"Next"**
4. 最后点击 **"Install"**
5. 安装完成后点击 **"Finish"**

### 1.3 验证安装

1. 按 `Win + R`
2. 输入 `cmd`
3. 按 `Enter`
4. 输入：`git --version`
5. 应该显示：`git version 2.43.0.windows.1`

✅ 如果显示版本号，说明安装成功！

---

## 🔑 步骤 2：获取 GitHub Token（3分钟）

### 2.1 访问 Token 设置页面

**直接访问**：https://github.com/settings/tokens

### 2.2 生成新 Token

1. 点击 **"Generate new token"**
2. 选择 **"Generate new token (classic)"**

### 2.3 配置 Token

填写以下信息：

**Note**（备注）：
```
qushuiyin-deploy
```

**Expiration**（过期时间）：
```
No expiration（永不过期）
```

**Select scopes**（权限）：
- ✅ 勾选 **repo**
  - 会自动勾选所有子选项：
    - repo:status
    - repo_deployment
    - public_repo
    - repo:invite
    - security_events

### 2.4 生成并保存 Token

1. 滚动到底部
2. 点击绿色按钮 **"Generate token"**
3. **立即复制 Token**（格式：`ghp_xxxxxxxxxxxxxxxxxxxx`）
4. **粘贴到记事本保存**（⚠️ 只显示一次！）

示例 Token 格式：
```
ghp_1234567890abcdefghijklmnopqrstuvwxyz12
```

## ⚙️ 步骤 3：配置 Git（2分钟）

### 3.1 打开命令行

1. 按 `Win + R`
2. 输入 `cmd`
3. 按 `Enter`

### 3.2 配置用户信息

输入以下命令（一行一行输入）：

```bash
git config --global user.name "aye0321123"
```

```bash
git config --global user.email "你的邮箱@example.com"
```

### 3.3 验证配置

```bash
git config --global user.name
```
应该显示：`aye0321123`

```bash
git config --global user.email
```
应该显示：您的邮箱

✅ 配置完成！

## 💾 保存 Token（可选，更方便）

### 方法：使用 Git Credential Manager

首次输入 Token 后，Git 会询问是否保存：
- 选择 **"是"** 或 **"Save"**
- 以后就不需要再输入了

或者手动配置：
```bash
git config --global credential.helper store
```

## 🔧 常见问题

### Q1: 安装 Git 后命令行仍提示找不到？
**A**: 关闭所有命令行窗口，重新打开

### Q2: Token 忘记保存了怎么办？
**A**: 
1. 访问：https://github.com/settings/tokens
2. 删除旧 Token
3. 重新生成新 Token

### Q3: 推送时提示认证失败？
**A**: 
1. 确认使用的是 Token，不是密码
2. 确认 Token 有 repo 权限
3. 重新生成 Token

### Q4: 每次都要输入 Token？
**A**: 
1. 运行：`git config --global credential.helper store`
2. 下次输入后会自动保存
