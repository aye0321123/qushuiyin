📌 最简单的方法：
   双击运行 → 自动上传到GitHub.bat

📌 让 Kiro 帮你上传，只需说：
   ✓ "上传到 GitHub"
   ✓ "帮我同步到网站"
   ✓ "更新到 GitHub Pages"

───────────────────────────────────────────────────────

🔑 首次使用需要：

1. GitHub 用户名：aye0321123

2. GitHub 密码：使用 Personal Access Token
   获取地址：https://github.com/settings/tokens
   
   创建步骤：
   - Generate new token (classic)
   - 勾选 repo 权限
   - 复制生成的 Token（ghp_xxxxx...）

3. 仓库地址格式：
   https://github.com/aye0321123/仓库名.git

───────────────────────────────────────────────────────

⚡ 完整工作流：

  修改工具 → 保存文件 → 运行上传脚本 → 等待 1-2 分钟 → 完成

───────────────────────────────────────────────────────

🌐 网站地址：
   https://aye0321123.github.io/仓库名/

📊 管理后台：
   https://aye0321123.github.io/仓库名/admin.html

───────────────────────────────────────────────────────

❓ 常见错误：

  "未检测到 Git" → 安装 Git 后重启命令行
  "需要密码" → 使用 Token，不是 GitHub 密码
  "上传失败" → 检查 Token 权限是否包含 repo
  "没有更改" → 确认文件已保存

───────────────────────────────────────────────────────

💡 提示：
   - Token 只显示一次，请妥善保存
   - Token 可以设置永不过期
   - 每次修改后都需要重新上传
   - 上传后等待 1-2 分钟才能看到更新
