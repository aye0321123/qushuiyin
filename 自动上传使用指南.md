# 🚀 自动上传到 GitHub 使用指南

## 📋 前提条件

### 1. 安装 Git
- 下载地址：https://git-scm.com/download/win
- 安装时选择默认选项即可
- 安装完成后重启命令行窗口

### 2. 配置 Git 用户信息（首次使用）
打开命令行，输入以下命令：

```bash
git config --global user.name "aye0321123"
git config --global user.email "你的邮箱@example.com"
```

### 3. 创建 GitHub Personal Access Token
1. 访问：https://github.com/settings/tokens
2. 点击 **Generate new token** → **Generate new token (classic)**
3. 设置：
   - Note: `视频解析工具自动上传`
   - Expiration: `No expiration`（永不过期）
   - 勾选权限：✅ **repo**（所有子选项）
4. 点击 **Generate token**
5. **重要**：复制生成的 Token（类似：`ghp_xxxxxxxxxxxxxxxxxxxx`）
6. 保存到安全的地方，后续上传时需要用它作为密码

## 🔧 常见问题

### Q1: 提示 "未检测到 Git"
**解决**：
1. 安装 Git：https://git-scm.com/download/win
2. 重启命令行窗口
3. 重新运行脚本

### Q2: 提示 "需要输入用户名和密码"
**解决**：
- 用户名：`aye0321123`
- 密码：使用 **Personal Access Token**（不是 GitHub 密码）
- Token 获取：https://github.com/settings/tokens

### Q3: 提示 "上传失败"
**可能原因**：
1. **Token 权限不足**：确保勾选了 `repo` 权限
2. **网络问题**：检查网络连接
3. **仓库地址错误**：检查仓库 URL 是否正确
4. **需要先拉取**：脚本会自动尝试 `git pull` 后重新推送

### Q4: 如何查看我的仓库地址？
1. 访问：https://github.com/aye0321123
2. 点击您的仓库
3. 点击绿色 **Code** 按钮
4. 复制 HTTPS 地址（例如：`https://github.com/aye0321123/qushuiyin.git`）

### Q5: 提示 "没有检测到文件更改"
**原因**：文件没有修改或已经上传过
**解决**：
- 确认文件已保存
- 如果确实有修改，可以手动运行：
  ```bash
  git add .
  git commit -m "更新"
  git push
  ```
