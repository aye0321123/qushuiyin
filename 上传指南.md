## 📤 上传方法

### 方法 1：FTP/SFTP 上传（推荐）

#### 使用 FileZilla：
1. 打开 FileZilla
2. 输入服务器信息：
   - 主机：您的服务器IP或域名
   - 用户名：FTP用户名
   - 密码：FTP密码
   - 端口：21（FTP）或 22（SFTP）
3. 连接成功后，找到网站根目录（通常是 `/www/wwwroot/您的域名/` 或 `/public_html/`）
4. 上传以下文件：
   - `index.html`
   - `admin.html`
   - `bg.jpg`

#### 使用 WinSCP：
1. 打开 WinSCP
2. 新建站点，输入服务器信息
3. 连接后拖拽文件到服务器目录

---

### 方法 2：宝塔面板上传

1. 登录宝塔面板：`http://您的服务器IP:8888`
2. 点击左侧菜单 **文件**
3. 进入网站目录（例如：`/www/wwwroot/您的域名/`）
4. 点击 **上传** 按钮
5. 选择并上传：
   - `index.html`
   - `admin.html`
   - `bg.jpg`
6. 如果已有旧文件，选择 **覆盖**

---

### 方法 3：cPanel 上传

1. 登录 cPanel 控制面板
2. 找到 **文件管理器**
3. 进入 `public_html` 目录
4. 点击 **上传**
5. 选择文件上传
6. 覆盖旧文件

---

### 方法 4：命令行上传（Linux/Mac）

```bash
# 使用 SCP 上传
scp index.html admin.html bg.jpg 用户名@服务器IP:/网站目录/

# 或使用 SFTP
sftp 用户名@服务器IP
put index.html
put admin.html
put bg.jpg
```

## 🔧 常见问题

### 问题 1：页面显示空白
**解决方案**：
- 检查文件是否上传成功
- 检查文件名是否正确（区分大小写）
- 查看浏览器控制台错误（按 F12）

### 问题 2：背景图片不显示
**解决方案**：
- 确认 `bg.jpg` 已上传
- 检查文件名大小写
- 检查文件路径是否正确

### 问题 3：平台图标不显示
**解决方案**：
- 这是正常的，图标使用 CDN 加载
- 如果 CDN 被墙，图标可能不显示
- 可以使用 emoji 替代（已在代码中备用）

### 问题 4：AI 改写不工作
**解决方案**：
- 检查 AI API 服务器是否在线
- 如果使用 HTTPS 网站，需要将 AI API 改为 HTTPS
- 或者禁用 AI 改写功能

### 问题 5：视频解析失败
**解决方案**：
- 检查 API Key 是否正确
- 确认 API 服务器正常
- 查看浏览器控制台错误信息
