## 🎯 部署步骤（跟着做）

### 步骤 1：创建 GitHub 仓库

1. **打开浏览器，访问**：
   ```
   https://github.com/new
   ```

2. **填写仓库信息**：
   - **Repository name**（仓库名）：`qushuiyin`（或任意名称）
   - **Description**（描述）：`视频去水印解析工具`
   - **Public**（公开）：✅ 选择这个
   - **Add a README file**：❌ 不要勾选（我们已经有了）

3. **点击绿色按钮**：`Create repository`

---

### 步骤 2：上传文件

创建仓库后，您会看到一个空仓库页面。

#### 方法 A：网页上传（推荐）

1. **点击**：`uploading an existing file`（上传现有文件）

2. **拖拽文件到页面**：
   - 打开文件夹：`C:\Users\Administrator.SY-202410161910\qushuiyin`
   - 选择以下文件拖到浏览器：
     - ✅ `index.html`
     - ✅ `admin.html`
     - ✅ `bg.jpg`
     - ✅ `README.md`
     - ✅ 所有 `.md` 文件
     - ✅ 所有 `.svg` 文件
     - ✅ 所有 `.bat` 文件

3. **填写提交信息**：
   - Commit message: `初始化视频解析工具`

4. **点击**：`Commit changes`

#### 方法 B：使用 GitHub Desktop（如果您有）

1. 下载 GitHub Desktop：https://desktop.github.com/
2. 克隆仓库到本地
3. 复制文件到仓库文件夹
4. 提交并推送

---

### 步骤 3：启用 GitHub Pages

1. **在仓库页面，点击**：`Settings`（设置）

2. **左侧菜单找到**：`Pages`

3. **配置 GitHub Pages**：
   - **Source**（来源）：`Deploy from a branch`
   - **Branch**（分支）：选择 `main`
   - **Folder**（文件夹）：选择 `/ (root)`

4. **点击**：`Save`

5. **等待部署**：
   - 页面会显示：`Your site is live at https://aye0321123.github.io/qushuiyin/`
   - 如果没有立即显示，刷新页面

---

### 步骤 4：访问网站

1. **等待 1-2 分钟**（GitHub Pages 需要构建）

2. **访问您的网站**：
   ```
   https://aye0321123.github.io/qushuiyin/
   ```

3. **访问管理后台**：
   ```
   https://aye0321123.github.io/qushuiyin/admin.html
   ```

## 🔧 常见问题

### Q1: 创建仓库时提示仓库名已存在

**解决**：
- 使用其他名称，如：`video-parser`、`qsy-tool` 等
- 或删除旧仓库后重新创建

### Q2: 上传文件后看不到

**解决**：
- 刷新页面
- 检查是否在正确的仓库

### Q3: Pages 设置中没有 "Save" 按钮

**解决**：
- 确保已经上传了文件
- 确保选择了 Branch 和 Folder

### Q4: 访问网站显示 404

**可能原因**：
1. **还在部署中**：等待 2-3 分钟
2. **文件位置错误**：
   - 如果文件在子文件夹，访问：`https://aye0321123.github.io/qushuiyin/子文件夹名/`
3. **Pages 未启用**：检查 Settings → Pages 是否配置正确

### Q5: 网站显示但样式错误

**解决**：
1. 按 `Ctrl + F5` 强制刷新
2. 清除浏览器缓存
3. 检查 `bg.jpg` 是否上传成功
