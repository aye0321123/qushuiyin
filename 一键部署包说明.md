## 🚀 最简单的部署方法（3 步完成）

### 方法一：拖拽上传（最简单！）

#### 步骤 1：创建仓库
1. 访问：https://github.com/new
2. 仓库名输入：`qushuiyin`
3. 选择 **Public**
4. 点击 **Create repository**

#### 步骤 2：上传整个文件夹
1. 在新建的仓库页面，点击：**uploading an existing file**
2. 打开文件夹：`C:\Users\Administrator.SY-202410161910\qushuiyin`
3. **按 Ctrl + A 全选所有文件**
4. **拖拽到浏览器**
5. 点击 **Commit changes**

#### 步骤 3：启用 Pages
1. 点击 **Settings** → **Pages**
2. Branch 选择：**main**
3. Folder 选择：**/ (root)**
4. 点击 **Save**

✅ **完成！** 等待 1-2 分钟后访问：
```
https://aye0321123.github.io/qushuiyin/
```
