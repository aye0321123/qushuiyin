🚀 最简单的部署方法（3分钟，无需安装任何软件）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第一步：复制本地文件内容
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 在当前文件夹找到 index.html
2. 右键 → 打开方式 → 记事本
3. 按 Ctrl+A 全选
4. 按 Ctrl+C 复制

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第二步：打开 GitHub 编辑页面
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在浏览器中访问（复制这个网址）：

https://github.com/aye0321123/qushuiyin/edit/main/qushuiyin/index.html

（如果提示登录，请先登录您的 GitHub 账号）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第三步：替换内容
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

在 GitHub 编辑器中：
1. 按 Ctrl+A 全选所有内容
2. 按 Delete 删除
3. 按 Ctrl+V 粘贴刚才复制的内容

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第四步：提交更改
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 滚动到页面底部
2. 在 "Commit message" 框中输入：
   优化移动端显示效果
3. 点击绿色按钮 "Commit changes"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第五步：等待部署并测试
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 等待 1-2 分钟
2. 访问您的网站：
   https://aye0321123.github.io/qushuiyin/qushuiyin/
3. 在手机上测试移动端效果

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 完成！

这个方法：
• 不需要安装 Git
• 不需要 Token
• 只需要 GitHub 账号登录
• 3 分钟完成

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

如果以后想要自动化部署，可以：
1. 手动下载 Git：https://git-scm.com/download/win
2. 安装完成后重启电脑
3. 再运行自动部署脚本

但现在用网页编辑是最快的！
